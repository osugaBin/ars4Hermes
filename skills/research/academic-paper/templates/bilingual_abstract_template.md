# Bilingual Abstract Template

## English abstract

```
## Abstract

[Background: 1-2 sentences. Context and the problem.]
[Purpose: 1 sentence. Research objective.]
[Method: 1-2 sentences. Approach and data.]
[Findings: 2-3 sentences. Key results with specific details.]
[Implications: 1-2 sentences. Significance and contribution.]

**Keywords**: [keyword1], [keyword2], [keyword3], [keyword4], [keyword5], [keyword6]
```

## Chinese abstract (中文摘要)

```
## 摘要

[背景：1-2 句。研究背景与问题。]
[目的：1 句。研究目标。]
[方法：1-2 句。研究方法。]
[发现：2-3 句。主要发现及具体数据。]
[意义：1-2 句。研究意义。]

**关键词：** [关键词1], [关键词2], [关键词3], [关键词4], [关键词5]
```

## Notes
- English: 150-250 words. Keywords: 5-7.
- Chinese: 200-300 characters. Keywords: 3-5.
- Content should be parallel (same claims, same structure)
- Do NOT include citations in the abstract body
- Write in third person, past tense for methods and results
