# Revision Tracking Template

Use when revising a paper based on reviewer comments.

## Revision tracking table

| # | Reviewer comment | Section affected | Action taken | Status |
|---|-----------------|-----------------|--------------|--------|
| 1 | "Add more recent lit" | Introduction | Added 3 refs from 2023-2025 | ✅ Done |
| 2 | "Clarify sampling" | 3.2 Methods | Rewrote with explicit criteria | ✅ Done |
| 3 | "Discuss limitation X" | 5.3 | Added paragraph | ✅ Done |
| 4 | "Expand conclusion" | 6 | Added practical implications | ✅ Done |

## Revision markers in the annotated draft

Use `[REVISED: ...]` markers inline to show what changed:

```
[REVISED: Added 3 references from 2023-2025 per reviewer comment #1]

[REVISED: Rewrote sampling criteria section per reviewer comment #2 — now explicit inclusion/exclusion]
```

## Output files

Produce two versions:
- `paper_final.md` — clean version (no markers)
- `paper_revision_track.md` — annotated version with `[REVISED]` markers
